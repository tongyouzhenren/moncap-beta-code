注意，此项目还没有完成，请勿使用

本项目采用nrf5340作为主控，lsm6dsv16x作为数据采集，通过专有esb协议以超低延迟将数据发送到中转mcu，再通过虚拟串口发送到pc。
下位机对采集的初始数据使用vqf算法进行初步滤波得到四元数，pc端使用gtsam库进行因子图创建，然后参考vip论文再进行姿态校准

使用方法：
需要安装zephyr环境，将源代码克隆到本地，将moncap的代码通过zephyr编译成可烧录文件，将server烧录到采集端，master烧录到中转端，编译时请使用多核编译指令。
请先确保打开server，之后通过拉低master端了引脚0.14电压发送初始化信号，初始化时确保身体姿态是T-Pose，维持10s左右，之后就会开始采集。
以warudo为例，运行仓库里的C#发行版程序，之后应用会自动处理数据转发。
如果想发送到别的渲染端，请更改代码里的udp端口和地址。
